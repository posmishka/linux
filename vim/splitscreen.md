splitscreen
=====
**Разделить по горизонтали**
	
		:sp <filepath>
		Ctrl-W s
		:split
		запуск vim -o file1 file2
		
**Разделить по вертикали**

		:vs <filepath>
		Ctrl-W v
		:verticalsplit
		запуск vim -O file1 file2
		
**Переключение**

		Ctrl-W, потом стрелками
		Ctrl-W, потом h j k l
		Ctrl-W,  w / W вперёд / назад
		
**Закрыть текущее окно**
		Ctrl-W c
		
	Закрыть всё кроме текущего
	
		Ctrl-W o
